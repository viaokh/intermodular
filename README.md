Intermodular 2025-2026
Importar el proyecto Maven en Eclipse
Clonar el proyecto un una ruta accesible (como el workplace de Eclipse).
Abrir Eclipse IDE o Eclipse Spring IDE.
Ir a File → Import.
Seleccionar Maven → Existing Maven Projects y pulsar Next.
En Root Directory, pulsar Browse y seleccionar la carpeta raíz del proyecto (la subcarpeta intermodular_260312).
Esperar a que Eclipse detecte el proyecto en la lista Projects.
Expandir Advanced.
En caso de conflicto, en Name template, escribir un nombre diferente para evitar el conflicto (uno diferente a peliculas-v4).
Verificar que el proyecto aparece seleccionado en la lista.
Pulsar Finish y esperar a que Eclipse descargue las dependencias de Maven y configure el proyecto automáticamente.
